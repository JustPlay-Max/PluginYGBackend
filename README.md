# PluginYG Backend

Серверная программа для сервисов PluginYG.
Сейчас backend используется для модуля покупок: каталог товаров, обработка заказов, платежные callback от платформ и выдача покупок в игру.

## Возможности

- Каталог товаров для PluginYG модуля Payments.
- Жизненный цикл заказа: `pending`, `paid`, `consumed`, `failed`.
- Поддержка платежных уведомлений Yandex Games, VK Games.
- Заготовка для платежных уведомлений Одноклассников, CrazyGames / Xsolla.
- Раздача картинок товаров через `/images/...`.
- Запуск как обычное Node.js приложение.
- Dockerfile для контейнерного размещения.

## Поддерживаемое размещение

Поддерживаются:

- Managed Node.js hosting, например Render или Railway.
- Docker hosting.
- VDS/VPS с Node.js, PM2, nginx и HTTPS.

Serverless/edge-сервисы вроде Cloudflare Workers, Netlify Functions и Vercel Functions пока не поддерживаются. Для них нужен отдельный runtime adapter и другое хранилище заказов.
Зарубежные managed-сервисы удобны для тестов, но для российской аудитории могут быть нестабильны из-за блокировок, замедлений, санкционных ограничений... Для релиза на российскую аудиторию рекомендуется VDS/VPS с хорошей доступностью из России.
